# Club Finder
A simple app to find a sport club using ES6, Web Components ( Custom Element & Shadow DOM ), NPM, Webpack, and SportDB API ( https://www.thesportsdb.com/api.php )

# Installation
1. Clone this repository
2. Run 
```
$ npm install
```
3. Run for development
```
$ npm run start-dev
```
   or for production
```
$ npm run build
```

GO!
